# ESP32-S3 iPod Music Player

A custom-built portable music player with modern touch interface and physical controls.

## Hardware Specifications

- **MCU**: ESP32-S3 (16MB Flash, 8MB PSRAM)
- **Display**: 2.8" ILI9341 capacitive touch (240x320px)
- **Storage**: 256GB microSD card
- **Audio**: USB-C (PCM2912A codec) + 3.5mm jack
- **Power**: USB-C PD (IP2723T) + 18650 LiPo (3500mAh)
- **Input**: Rotary encoder (KY-040) + capacitive touch
- **Connectivity**: Bluetooth 5.0 A2DP

## Key Features

- USB-C Power Delivery (15W charging)
- Simultaneous charging and digital audio output
- Bluetooth wireless streaming
- Touch + rotary encoder navigation
- 24-bit/96kHz audio quality
- 6-8 hours battery life
- OTA updates

## Development Phases

1. **Phase 1**: Core Hardware Setup
2. **Phase 2**: Basic Software (Display, SD, Audio, Encoder)
3. **Phase 3**: Advanced Features (USB-C, Bluetooth, Touch)
4. **Phase 4**: Polish (Album Art, Animations, EQ)
